# LJT-Homepage

Personal academic homepage for Junteng Liu.

## About

This is the personal academic homepage of Junteng Liu, a first-year PhD candidate at HKUST NLP Group.

## Setup

1. Clone this repository
2. Install Jekyll: `gem install jekyll bundler`
3. Install dependencies: `bundle install`
4. Run locally: `bundle exec jekyll serve`
5. Open your browser at `http://localhost:4000/LJT-Homepage/`

## Structure

- `_config.yml` - Main Jekyll configuration
- `_pages/` - Main pages (About, Publications, CV)
- `_publications/` - Individual publication pages
- `_data/` - Data files (navigation, authors, UI text)
- `_layouts/` - HTML layout templates
- `_includes/` - Reusable HTML components
- `assets/` - CSS, JS, and other assets
